# CardGame

Description:
---------------

This game has the following 5 options:

  OPTION 1: Get a shuffled deck of cards
  OPTION 2: Draw the top card
  OPTION 3: Sort the cards by a given color list
  OPTION 4: Play a game
  OPTION 5: Exit
 
How to run:
---------------
  java -jar essex-card-game.jar
